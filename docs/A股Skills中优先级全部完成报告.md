## 本次完成的Skills详细说明

### 7. ipo-lockup-risk-monitor（IPO解禁风险监控）

**核心功能**：监控IPO解禁风险，评估解禁对股价的影响

**关键指标**：
- 解禁规模 = 解禁股数 × 当前股价
- 解禁压力 = 解禁规模 / 流通市值 × 100%
- 解禁收益率 = (当前价 - 成本价) / 成本价 × 100%
- 减持意愿 = f(解禁收益率, 股东类型, 市场环境)

**可测试规则**：
1. 大规模解禁 + 高收益率 → 减持压力大
2. 创投解禁 > 产业资本减持意愿
3. 解禁前1个月股价下跌 → 减持压力释放
4. 小盘股解禁冲击 > 大盘股

**A股特殊性**：
- 减持新规（大宗交易、竞价交易限制）
- 解禁类型多样（首发、定增、股权激励）
- 减持公告提前披露
- 窗口期限制

### 8. event-study（事件研究）

**核心功能**：分析特定事件对股价的影响，量化事件的超额收益

**关键指标**：
- 异常收益 AR = 实际收益 - 预期收益
- 累计异常收益 CAR = Σ(AR) over event window
- t统计量 = CAR / SE(CAR)
- 事件冲击 = CAR[0, +1]

**可测试规则**：
1. 正面事件 + 显著正CAR → 买入机会
2. 负面事件 + 显著负CAR → 卖出信号
3. 事件前异常上涨 → 信息泄露
4. 小盘股事件效应 > 大盘股

**A股特殊性**：
- T+1交易延迟
- 涨跌停限制（异常收益截断）
- 停牌影响（复牌后补涨/补跌）
- 信息泄露常见

**特别说明**：本次修复了event-study文档的损坏内容，恢复了完整的methodology结构。

### 9. dividend-corporate-action-tracker（分红与公司行动跟踪器）

**核心功能**：跟踪分红派息、送转股、配股、增发等公司行动

**关键指标**：
- 股息率 = 每股分红 / 股价 × 100%
- 分红率 = 分红总额 / 归母净利润 × 100%
- 分红连续性 = 连续分红年数
- 分红稳定性 = 1 - std(分红率) / mean(分红率)

**可测试规则**：
1. 高股息率 + 稳定分红 → 防御性配置
2. 分红率突然大幅提升 → 警惕业绩见顶
3. 高送转 + 低现金分红 → 警惕炒作
4. 分红 + 回购 → 股东友好信号

**A股特殊性**：
- 红利税（持股期限影响）
- 高送转炒作（A股特色）
- 再融资分红要求
- 大股东分红后减持

### 10. ipo-newlist-monitor（IPO新股监控）

**核心功能**：监控IPO新股上市，评估申购价值和破发风险

**关键指标**：
- 发行市盈率 = 发行价 / 每股收益
- 估值溢价率 = (发行PE - 行业PE) / 行业PE × 100%
- 首日涨幅 = (首日收盘价 - 发行价) / 发行价 × 100%
- 市场破发率 = 近期破发新股数 / 近期上市新股数 × 100%

**可测试规则**：
1. 低估值 + 低中签率 → 高申购价值
2. 高估值 + 大规模发行 → 破发风险
3. 科创板/创业板 > 主板首日涨幅
4. 市场破发率 > 30% → 暂停申购

**A股特殊性**：
- 注册制改革（破发常态化）
- 涨跌停限制差异（主板44%，科创板/创业板无限制）
- 中签率差异（主板低，科创板高）
- 市场情绪影响大

### 11. etf-allocator（ETF配置器）

**核心功能**：构建ETF组合，实现低成本、分散化的资产配置

**关键指标**：
- 跟踪误差 = std(ETF收益率 - 指数收益率) × √252
- 折溢价率 = (ETF价格 - ETF净值) / ETF净值 × 100%
- 夏普比率 = (年化收益率 - 无风险利率) / 年化波动率
- 分散度 = 1 - Σ(权重^2)

**可测试规则**：
1. 宽基ETF核心配置 → 长期稳健收益
2. 行业ETF轮动 → 超额收益
3. 折价ETF套利 → 低风险收益
4. 低波动ETF防御 → 下跌保护

**A股特殊性**：
- T+1交易（无法日内套利）
- 申赎门槛高（100万份）
- 规模风险（小规模ETF清盘风险）
- 跨境ETF QDII额度限制

### 12. rebalancing-planner（再平衡规划器）

**核心功能**：规划组合再平衡策略，维持目标配置，控制风险

**关键指标**：
- 偏离度 = |当前权重 - 目标权重|
- 再平衡需求 = 目标市值 - 当前市值
- 交易成本 = 交易金额 × (佣金率 + 印花税率 + 过户费率)
- 再平衡效率 = 再平衡收益 / 再平衡成本

**可测试规则**：
1. 定期再平衡 → 长期超额收益
2. 阈值触发再平衡 > 定期再平衡
3. 风险平价再平衡 → 降低波动
4. 税收优化再平衡 → 提升税后收益

**A股特殊性**：
- T+1交易（再平衡需要2天）
- 涨跌停限制（无法完成再平衡）
- 交易成本（佣金+印花税+过户费）
- 红利税（持股期限影响）
