# 常用“页面信号”方法论提炼（数据源无关）

本文件把常见金融数据页面（排名/持仓/交易/事件等）背后的分析方法，提炼为**可复盘、可交叉验证**的规则框架；数据可来自任意来源（第三方平台、交易所披露、开源数据等）。

核心原则：**单一页面指标只做“线索”，必须用资金/事件/可交易性做交叉验证。**

---

## 1) 注意力/情绪（关注度、热度、人气榜）

**常见字段**：总关注、日/周/月新增关注、热度分值/排名、评论/投票（若有）。

**常见用法**
- 热点轮动：Top N 的行业/概念聚集度上升，且新增关注持续为正 → 主题扩散
- 真假突破：热度↑ + 价格突破 + 成交温和放大 → 更可信；热度↓ + 价格突破 → 更像假突破
- 过热风险：热度创高但价格滞涨/冲高回落 → 拥挤风险提示（非确定顶部）

**对应技能**
- `$hot-rank-sentiment-monitor`（方法论已补齐“关注度/热度/背离/轮动”框架）
- 交叉验证：`$fund-flow-monitor`、`$concept-board-analyzer`、`$industry-board-analyzer`、`$disclosure-notice-monitor`

---

## 2) 筹码结构（人均持股、股东户数、户数变化排名）

**常见字段**：股东户数、户数变化/变化率、人均持股（若有）、前十大股东/流通股东、机构持仓（若有）。

**常见用法**
- 集中/收集线索：户数下降 + 人均上升 + 连续性（≥3期）→ 集中趋势更可信
- 派发/扩散风险：户数上升（尤其暴增）+ 高位滞涨/回撤加深 → 风险更高
- 关键是“连续性”而非单期：单期变化容易被财报窗口、解禁、股本结构变化干扰

**对应技能**
- `$shareholder-structure-monitor`（方法论已补齐“户数/人均/连续性/粗略代理/失效场景”）
- 交叉验证：`$ipo-lockup-risk-monitor`、`$equity-pledge-risk-monitor`、`$fund-flow-monitor`、`$liquidity-impact-estimator`

---

## 3) 杠杆资金（融资融券/两融）

**常见字段**：融资余额、融资买入额（或净买入）、融券余量/卖出量（若有）。

**常见用法**
- 趋势支持：融资余额连续上升 + 价格趋势向上 → 风险偏好偏强
- 顶背离风险：融资余额创新高但价格不创新高 → 拥挤与回撤风险提示
- 去杠杆踩踏：融资余额快速下降 + 波动放大 + 制度约束（涨跌停/停牌）→ 需要强调“不可交易风险”

**对应技能**
- `$margin-risk-monitor`（方法论已补齐“背离/过热/踩踏/融券边界”）
- 交叉验证：`$volatility-regime-monitor`、`$limit-up-limit-down-risk-checker`、`$liquidity-impact-estimator`

---

## 4) 换手/流动性（可交易性与冲击成本）

**常见字段**：成交额、换手率、成交稳定性、价差/盘口深度（若有）。

**常见用法**
- 低流动性风险：长期低换手/低成交额 → 大仓位难进难出
- 过热活跃：短期换手/成交到历史高分位 + 波动放大 → 情绪脉冲/滑点风险
- 规模约束：用 `OrderValue / ADV20` 作为第一层风控指标

**对应技能**
- `$liquidity-impact-estimator`（方法论已补齐“SizeToADV/冲击代理/分层/降级策略”）
- 日内结构辅助：`$intraday-microstructure-analyzer`

---

## 5) 波动率（风险状态机、仓位与止损尺度）

**常见字段**：日/周/月波动、ATR、回撤、波动分位。

**常见用法**
- 状态划分：用历史分位把波动分成低/中/高（避免写死阈值）
- 波动放大：事件冲击/趋势加速窗口（不预测方向）
- 波动收缩：方向选择窗口（提醒盯突破/破位）

**对应技能**
- `$volatility-regime-monitor`（方法论已补齐“分位状态机/仓位止损示例/交叉验证”）

---

## 6) 收益率排名 / 动量

**常见字段**：1M/3M/1Y/3Y 收益率、年化、风险调整收益（若有）。

**常见用法**
- 多周期一致性：短中长期都强的标的更“稳”，但需防拥挤与回撤
- 反转提示：短期强但长期弱，可能只是反弹（需事件与资金验证）

**对应技能**
- `$quant-factor-screener`（动量因子与“跳过最近月”的口径已内置在方法论中）
- 风控辅助：`$volatility-regime-monitor`、`$liquidity-impact-estimator`

---

## 7) “神奇公式”类（价值 + 质量的规则化筛选）

**核心思想**：高盈利收益率（Earnings Yield）+ 高资本回报（ROIC/ROE 等）。

**对应技能**
- `$quant-factor-screener`（价值因子 + 质量因子组合）
- `$financial-statement-analyzer`（排除一次性利润/现金流质量差/高杠杆等“便宜有原因”）

---

## 8) 事件与公告（财报预告、回购增持、解禁减持等）

**常见用法**
- 事件清单化：事件 → 潜在影响路径 → 不确定性 → 证伪条件
- 回购/增持：信号强度分层（金额/比例、执行节奏、用途、估值背景）
- 解禁/减持：供给冲击评估（解禁比例、持有人类型、浮盈、市况）

**对应技能**
- `$event-driven-detector`（已包含回购增持、解禁减持等框架）
- `$disclosure-notice-monitor`（公告与披露事件清单）
- `$ipo-lockup-risk-monitor`（解禁）

---

## 9) 风险红旗（质押、ST/退市等）

**对应技能**
- `$equity-pledge-risk-monitor`（质押）
- `$st-delist-risk-scanner`（ST/退市）
- `$shareholder-risk-check`（治理/股东层风险汇总）
