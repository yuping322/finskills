## 文档补充标准

根据US-market的92%质量标准，每个methodology文档应包含：

### 必需内容
1. **数据定义**（10分）
   - 明确列出所有使用的AKShare数据源
   - 说明每个数据源的字段和含义
   - 数据更新频率和延迟

2. **核心指标**（20分）

   - 指标的取值范围和单位

3. **可测试规则**（30分）
   - 至少4条规则
   - 每条规则包含：
     - IF条件（具体阈值）
     - THEN结论
     - CONFIDENCE置信度（格式：0.XX; 初步估计；需要历史验证）
     - APPLICABLE_UNIVERSE适用范围
     - FAILURE_MODE失效模式

4. **A股特性**（20分）
   - 至少10个方面的A股市场特殊性
   - T+1、涨跌停、停牌、监管等

5. **边界条件**（10分）
   - 数据缺失处理
   - 异常值处理
   - 降级策略

6. **回测说明**（10分）
   - 建议的回测方法
   - 需要验证的假设
   - 历史数据要求

## 补充计划

### 第1步：block-deal-monitor（大宗交易监控）

**当前问题**：
- 缺少具体的折价率阈值
- 没有明确的连续性判断标准
- 缺少数据源定义

**补充内容**：
```markdown
## 数据定义

### 主要数据源
1. **stock_dzjy_mrmx**（大宗交易每日明细）
   - 来源：AKShare - 东方财富
   - 字段：交易日期、股票代码、股票名称、成交价、成交量、成交额、买方营业部、卖方营业部
   - 更新：T+1日更新前一交易日数据
   - 延迟：约1-2小时

2. **stock_zh_a_hist**（A股历史行情）
   - 用途：计算折溢价率
   - 字段：日期、开盘、收盘、最高、最低、成交量、成交额
   - 更新：T+1日更新

## 核心指标

### 1. 折溢价率
**公式**：
```
折溢价率 = (大宗成交价 - 当日收盘价) / 当日收盘价 * 100%
```
**取值范围**：-20% ~ +20%（极端情况可能超出）
**单位**：百分比

### 2. 成交金额占比
**公式**：
```
成交金额占比 = 大宗成交额 / 当日总成交额 * 100%
```
**取值范围**：0% ~ 100%
**单位**：百分比

### 3. 连续交易天数
**定义**：连续N个交易日内出现大宗交易的天数
**取值范围**：1 ~ 20天
**单位**：天

### 4. 累计成交金额
**公式**：
```
累计成交金额 = Σ(连续N日的大宗成交额)
```
**取值范围**：0 ~ 无上限
**单位**：元

### 5. 平均折价率
**公式**：
```
平均折价率 = Σ(每日折价率 * 成交额) / Σ(成交额)
```
**取值范围**：-20% ~ +20%
**单位**：百分比

## 可测试规则

### 规则1：建仓型信号
**IF条件**：
- 折价率在 -5% ~ -2% 之间（温和折价）
- 连续3天以上出现大宗交易
- 平均折价率变化 < 1%（成交均价稳定）
- 二级市场换手率上升 > 20%

**THEN结论**：
可能存在机构建仓行为，建议关注

**CONFIDENCE**：
0.65; 初步估计；需要历史验证

**APPLICABLE_UNIVERSE**：
- 流通市值 > 50亿
- 日均成交额 > 5000万
- 非ST股票

**FAILURE_MODE**：
- 大宗交易方为股东减持
- 市场整体下跌导致折价
- 股权激励等特殊事件

### 规则2：抛压型信号
**IF条件**：
- 折价率 < -7%（显著折价）
- 成交金额占比 > 10%
- 二级市场承接弱（换手率下降 > 10%）
- 与解禁/减持窗口重合（前后5个交易日）

**THEN结论**：
存在较大抛压风险，建议规避

**CONFIDENCE**：
0.75; 初步估计；需要历史验证

**APPLICABLE_UNIVERSE**：
- 所有A股
- 特别关注解禁期股票

**FAILURE_MODE**：
- 市场情绪极度乐观时，抛压被快速消化
- 大宗买方为战略投资者，后续不再抛售

### 规则3：溢价成交警示
**IF条件**：
- 溢价率 > 5%
- 连续2天以上溢价成交
- 二级市场股价不跟随上涨（涨幅 < 2%）

**THEN结论**：
可能存在"利好出尽"风险，建议谨慎

**CONFIDENCE**：
0.60; 初步估计；需要历史验证

**APPLICABLE_UNIVERSE**：
- 所有A股
- 特别关注热门概念股

**FAILURE_MODE**：
- 溢价成交为股权激励等特殊事件
- 后续有重大利好公告

### 规则4：流动性风险
**IF条件**：
- 日均成交额 < 5000万
- 大宗成交额 > 日均成交额 * 50%
- 折价率 < -5%

**THEN结论**：
流动性风险较高，二次供给压力大

**CONFIDENCE**：
0.70; 初步估计；需要历史验证

**APPLICABLE_UNIVERSE**：
- 小盘股（流通市值 < 50亿）
- 成交不活跃股票

**FAILURE_MODE**：
- 大宗买方为长期投资者
- 市场流动性突然改善

## A股市场特性

### 1. T+1交易制度
- 大宗交易买方T+1才能卖出
- 影响：买方需承担隔夜风险，要求更高折价

### 2. 涨跌停限制
- 主板/中小板/创业板：±10%
- 科创板/创业板注册制：±20%
- 北交所：±30%
- 影响：折价率受涨跌停限制影响

### 3. 大宗交易规则
- 单笔成交量 ≥ 30万股 或 成交金额 ≥ 200万元
- 交易时间：15:00-15:30
- 成交价格：±10%（主板）或±20%（科创板）
- 影响：折价率有上下限

### 4. 减持新规
- 大股东减持需提前公告
- 集中竞价减持有数量限制
- 大宗交易减持后6个月内不得转让
- 影响：大宗交易可能是规避减持限制的手段

### 5. 解禁压力
- 定增解禁、首发解禁、股权激励解禁
- 解禁前后大宗交易活跃
- 影响：解禁期附近的大宗交易风险更高

### 6. 数据延迟
- 大宗交易数据T+1日公布
- 买卖方营业部信息可能不完整
- 影响：信号滞后，需结合二级市场表现

### 7. 营业部识别
- 机构专用席位：通常为基金、券商自营
- 知名游资席位：如"东方财富证券拉萨团结路"
- 影响：买卖方性质影响信号可靠性

### 8. 停牌影响
- 停牌期间无法大宗交易
- 复牌后可能集中出现大宗交易
- 影响：需关注停牌前后的大宗交易

### 9. 股权激励
- 股权激励行权后可能通过大宗交易减持
- 通常有锁定期限制
- 影响：需区分股权激励相关的大宗交易

### 10. 市场情绪
- 牛市中折价率普遍较小
- 熊市中折价率普遍较大
- 影响：需结合市场环境调整阈值

## 边界条件处理

### 数据缺失
- **买卖方信息缺失**：降级为仅分析价格和数量
- **历史行情缺失**：无法计算折溢价率，跳过该记录
- **成交额为0**：数据异常，过滤该记录

### 异常值处理
- **折价率 > 20% 或 < -20%**：标记为异常，单独分析
- **成交量异常大**：可能是协议转让，单独标记
- **连续涨停/跌停**：折价率失真，添加警告

### 降级策略
- **AKShare接口失败**：使用缓存数据（最多1天）
- **数据不完整**：仅输出可计算的指标
- **计算失败**：返回空结果并记录错误

## 回测建议

### 回测方法
1. **样本选择**：
   - 时间范围：2020-2023年（3年数据）
   - 股票池：全A股
   - 过滤：剔除ST、停牌超过30天的股票

2. **信号验证**：
   - 建仓型信号：统计后续20日收益率分布
   - 抛压型信号：统计后续10日最大回撤
   - 溢价成交：统计后续5日涨跌幅

3. **阈值优化**：
   - 折价率阈值：测试-2%、-3%、-5%、-7%、-10%
   - 连续天数：测试2天、3天、5天
   - 成交占比：测试5%、10%、15%、20%

### 需要验证的假设
1. 温和折价（-2%~-5%）+ 连续性 = 建仓信号
2. 显著折价（<-7%）+ 高占比 = 抛压信号
3. 溢价成交 + 股价不涨 = 利好出尽
4. 小盘股 + 大额大宗 = 流动性风险

### 历史数据要求
- 大宗交易明细：至少3年
- 日线行情：至少3年
- 解禁数据：至少3年
- 公告数据：至少3年（用于识别特殊事件）
```
