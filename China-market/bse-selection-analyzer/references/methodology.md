# 方法论：北交所精选（以可交易性为核心）

## 数据定义

### 主要数据源
1. **stock_bj_a_spot_em**（北交所实时行情）
   - 来源：AKShare - 东方财富
   - 字段：股票代码、股票名称、最新价、涨跌幅、成交量、成交额、换手率、市盈率、市净率
   - 更新：实时更新（盘中）
   - 延迟：约3-5秒

2. **stock_bj_a_hist**（北交所历史行情）
   - 用途：计算日均成交额、波动率
   - 字段：日期、开盘、收盘、最高、最低、成交量、成交额
   - 更新：T+1日更新

3. **stock_financial_abstract_em**（财务摘要）
   - 用途：基本面筛选
   - 字段：营业收入、净利润、ROE、毛利率、资产负债率
   - 更新：季度更新（财报披露时）

## 核心指标

### 1. 日均成交额
**公式**：
```
日均成交额 = Σ(近20日成交额) / 20
```
**取值范围**：0 ~ 无上限
**单位**：万元
**阈值**：≥ 500万（流动性过滤）

### 2. 成交额稳定性
**公式**：
```
成交额稳定性 = std(近20日成交额) / mean(近20日成交额)
```
**取值范围**：0 ~ 无上限（越小越稳定）
**单位**：无量纲
**阈值**：< 1.5（避免单日放量幻觉）

### 3. 营收增长率
**公式**：
```
营收增长率 = (本期营收 - 上期营收) / 上期营收 * 100%
```
**取值范围**：-100% ~ 无上限
**单位**：百分比
**阈值**：≥ 20%（成长性标准）

### 4. 净利润增长率
**公式**：
```
净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润 * 100%
```
**取值范围**：-100% ~ 无上限
**单位**：百分比
**阈值**：≥ 15%（盈利增长标准）

### 5. 市盈率PE
**公式**：
```
PE = 股价 / 每股收益
```
**取值范围**：0 ~ 无上限
**单位**：倍
**阈值**：< 30（估值标准）

### 6. 市净率PB
**公式**：
```
PB = 股价 / 每股净资产
```
**取值范围**：0 ~ 无上限
**单位**：倍
**阈值**：< 3（估值标准）

### 7. ROE（净资产收益率）
**公式**：
```
ROE = 净利润 / 净资产 * 100%
```
**取值范围**：-100% ~ 无上限
**单位**：百分比
**阈值**：≥ 10%（盈利能力标准）

## 可测试规则

### 规则1：优质成长股
**IF条件**：
- 日均成交额 ≥ 500万
- 成交额稳定性 < 1.5
- 营收增长率 ≥ 20%
- 净利润增长率 ≥ 15%
- PE < 30
- ROE ≥ 10%

**THEN结论**：
优质成长股，流动性好，基本面优秀，建议关注

**CONFIDENCE**：
0.68; 初步估计；需要历史验证

**APPLICABLE_UNIVERSE**：
- 北交所精选层股票
- 流动性充足股票

**FAILURE_MODE**：
- 业绩不及预期（增长放缓）
- 估值中枢下移（市场风格切换）
- 流动性枯竭（成交萎缩）
- 监管政策变化（北交所规则调整）

### 规则2：低估值价值股
**IF条件**：
- 日均成交额 ≥ 500万
- PE < 15
- PB < 1.5
- ROE ≥ 10%
- 净利润增长率 ≥ 0%（盈利稳定）
- 资产负债率 < 60%

**THEN结论**：
低估值价值股，安全边际高，适合稳健投资

**CONFIDENCE**：
0.62; 初步估计；需要历史验证

**APPLICABLE_UNIVERSE**：
- 北交所精选层股票
- 传统行业、成熟企业

**FAILURE_MODE**：
- 行业衰退（盈利下滑）
- 估值陷阱（低估值有原因）
- 流动性风险（成交萎缩）
- 市场风格偏向成长（价值股被忽视）

### 规则3：流动性风险警示
**IF条件**：
- 日均成交额 < 500万
- 成交额稳定性 > 2.0（波动大）
- 换手率 < 1%（成交不活跃）

**THEN结论**：
流动性风险高，冲击成本大，建议规避

**CONFIDENCE**：
0.75; 初步估计；需要历史验证

**APPLICABLE_UNIVERSE**：
- 北交所所有股票
- 特别关注小盘股

**FAILURE_MODE**：
- 流动性突然改善（热点炒作）
- 公司基本面改善（吸引资金）
- 北交所整体流动性改善

### 规则4：高估值风险
**IF条件**：
- PE > 50
- PB > 5
- 净利润增长率 < 20%（增长不支撑高估值）
- 日均成交额 < 1000万（流动性一般）

**THEN结论**：
高估值风险，性价比低，建议谨慎

**CONFIDENCE**：
0.70; 初步估计；需要历史验证

**APPLICABLE_UNIVERSE**：
- 北交所所有股票
- 特别关注热门概念股

**FAILURE_MODE**：
- 业绩超预期（增长加速）
- 市场风格偏向成长（高估值被接受）
- 公司有重大利好（估值重估）

## A股市场特性

### 1. 北交所涨跌停限制
- 精选层：±30%（比主板更宽）
- 新股上市首日：不设涨跌幅限制
- 影响：波动更大，风险更高

### 2. 北交所交易规则
- 连续竞价：9:30-11:30, 13:00-14:57
- 集合竞价：14:57-15:00
- 最小报价单位：0.01元
- 影响：交易机制与主板不同

### 3. 北交所流动性特征
- 整体流动性较差（相比主板）
- 个股流动性分化严重
- 机构参与度较低
- 影响：需要严格流动性过滤

### 4. 北交所转板机制
- 精选层公司可申请转板至科创板或创业板
- 转板条件：市值、盈利、规范运作等
- 影响：转板预期是重要催化剂

### 5. 北交所投资者门槛
- 个人投资者：资产≥50万，2年交易经验
- 机构投资者：无门槛
- 影响：投资者结构与主板不同

### 6. 北交所公司特征
- 以中小企业为主
- 以创新型、成长型企业为主
- 行业分布：先进制造、现代服务、高技术服务
- 影响：成长性好但风险高

### 7. 北交所信息披露
- 披露要求与主板一致
- 中小企业信息披露质量参差不齐
- 影响：需要更谨慎的基本面分析

### 8. 北交所估值特征
- 整体估值高于主板（成长溢价）
- 估值分化严重（流动性溢价）
- 影响：需要结合成长性和流动性评估估值

### 9. 北交所政策支持
- 国家支持中小企业发展
- 北交所定位为服务创新型中小企业
- 影响：政策利好是重要支撑

### 10. 北交所风险特征
- 退市风险高于主板
- 业绩波动大
- 流动性风险高
- 影响：需要严格风控

## 边界条件处理

### 数据缺失
- **财务数据缺失**：使用业绩预告或业绩快报，标注"预估数据"
- **历史行情缺失**：无法计算日均成交额，跳过该股票
- **成交额为0**：停牌或数据异常，过滤该股票

### 异常值处理
- **PE < 0（亏损）**：使用PB或PS估值，标注"PE不适用"
- **ROE > 50%**：检查是否有非经常性损益，标注"ROE异常"
- **净利润增长率 > 200%**：检查是否有并购、资产注入，标注"增长异常"

### 降级策略
- **AKShare接口失败**：使用缓存数据（最多1天）
- **数据不完整**：仅输出可计算的指标，标注"数据不完整"
- **计算失败**：返回空结果并记录错误

## 回测建议

### 回测方法
1. **样本选择**：
   - 时间范围：2021-2023年（北交所设立后）
   - 股票池：北交所精选层
   - 过滤：剔除ST、停牌超过30天的股票

2. **信号验证**：
   - 优质成长股：统计后续60日收益率分布
   - 低估值价值股：统计后续90日收益率分布
   - 流动性风险：统计冲击成本和滑点

3. **阈值优化**：
   - 日均成交额：测试300万、500万、800万、1000万
   - 营收增长率：测试15%、20%、25%、30%
   - PE阈值：测试20、25、30、35

### 需要验证的假设
1. 日均成交额 ≥ 500万 = 流动性充足
2. 营收增长率 ≥ 20% + 净利润增长率 ≥ 15% = 优质成长
3. PE < 30 + PB < 3 = 估值合理
4. 日均成交额 < 500万 = 流动性风险

### 历史数据要求
- 北交所行情数据：至少2年（2021年至今）
- 财务数据：至少2年（8个季度）
- 转板数据：至少1年（用于分析转板预期）
- 流动性数据：至少1年（用于分析流动性变化）
